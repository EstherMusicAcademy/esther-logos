## 다운로드 방법

1. 위 링크 클릭 -> 학교 공식 UI 페이지 접속
2. 로고/심볼마크 이미지 다운로드 (PNG 권장)
3. 파일명: "학교명.png" (예: 경희대학교.png)
4. 이 폴더(대학로고/)에 저장

## 공식 페이지 없는 학교

- 구글 이미지 검색: "학교명 로고 png"
- 또는 학교 홈페이지 상단 로고 이미지 저장
- 또는 나무위키 등에서 로고 이미지 확인
